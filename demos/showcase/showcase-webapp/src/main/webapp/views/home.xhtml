Liferay Faces is an umbrella project that provides support for the JavaServerTM Faces (JSF) standard within Liferay Portal. This showcase application demonstrates usage of Facelet component tags provided by the following sub-projects:

The Liferay Faces Alloy sub-project provides a suite of JSF UI components that utilize Alloy UI.

The Facelet component tags are prefixed with the alloy: namespace.

The Liferay Faces Bridge sub-project includes a JSR 329 certified portlet bridge which provides the ability to deploy JSF 2.2 web applications as portlets within Liferay Portal.

The Facelet component tags are prefixed with the bridge: and portlet: namespaces.

The Liferay Faces Portal sub-project provides JSF utilities, EL keywords, and a suite of JSF UI components for use with Liferay Portal.

The Facelet component tags are prefixed with the portal: namespace.
